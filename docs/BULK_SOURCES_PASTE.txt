HOW TO USE THIS FILE (Your Tab-Based UI)
================================================================================

Your UI has TABS for each source type. Here's how to bulk add sources:

METHOD 1: Quick Copy-Paste (Fastest)
-------------------------------------
1. Go to: http://localhost:3000/app/settings → Click "Sources" in sidebar
2. Pick a tab (Reddit / RSS / Twitter / YouTube / Blogs)
3. Copy ONE LINE at a time from sections above
4. Paste into the input field
5. Press ENTER or click "Add"
6. Repeat for each source
7. Click "Save Sources" button at bottom when done

METHOD 2: Use Bulk Paste Script (Recommended for 20+ sources)
--------------------------------------------------------------
Instead of manually adding 45 sources one by one, use the API method:

1. Open Terminal/PowerShell
2. Run these commands:

   # Get your auth token (login)
   curl -X POST http://localhost:8000/api/v1/auth/login \
     -H "Content-Type: application/json" \
     -d "{\"email\": \"your@email.com\", \"password\": \"yourpassword\"}"

   # Copy the access_token from response
   # Set environment variables
   export TOKEN="paste-your-token-here"
   export WORKSPACE_ID="paste-your-workspace-id-here"

   # Save one of the configs from SECTION 4 above to a file
   # Then update workspace config:
   curl -X PUT http://localhost:8000/api/v1/workspaces/$WORKSPACE_ID/config \
     -H "Authorization: Bearer $TOKEN" \
     -H "Content-Type: application/json" \
     -d @config.json

3. Refresh your UI page to see all sources added

EXAMPLE: Adding 5 Reddit Subreddits via UI
-------------------------------------------
1. Go to /app/settings → Sources → Reddit tab
2. Copy "MachineLearning" from SECTION 1 above
3. Paste into "Start typing subreddit name..." field
4. Press ENTER (it adds as a badge)
5. Repeat for: artificial, LocalLLaMA, OpenAI, LangChain
6. Click "Save Sources" at bottom
7. Done! You'll see 5 badges showing your subreddits
